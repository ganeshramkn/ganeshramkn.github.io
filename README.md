## How to Deploy Using GitHub Pages

1.  Create a new repository on GitHub.
2.  Upload all files while maintaining the folder structure.
3.  Go to Repository Settings.
4.  Navigate to Pages.
5.  Under "Source", select:
    -   Deploy from branch
    -   Branch: main
6.  Save the settings.

Your website will be available at:

https://yourusername.github.io/repository-name/

Replace "yourusername" and "repository-name" with your actual GitHub
details.
